## ⚡️ Overview

- **Performance-First Architecture**: Built on Astro + React with a static-first approach and focused optimization for fast delivery.
- **Component-Driven Design**: fully typed React components with reusable UI primitives and atomic design principles.
- **Engineering Best Practices**: automated testing (Unit & E2E), strict TypeScript, linting, and static analysis.
- **User Experience**: Seamless bilingual support (ES/EN), dark/light mode synchronization, and accessible (`a11y`) navigation.

## ✨ Key Features

- **🌐 Internationalization (i18n)**: Robust, type-safe translation system with automatic browser language detection.
- **🌗 Theme System**: System-aware Dark/Light mode with persisted preferences logic in `nanostores`.
- **⚡️ Core Web Vitals**: Optimized for Lighthouse performance, accessibility, best practices, and SEO.
- **🔒 Privacy & Security**: Self-hosted fonts (`@fontsource/inter`) and secure defaults for production deployment.
- **📱 Responsive**: "Mobile-First" responsive design covering devices from 320px to 4k.

## 🏗 Project Structure

A clean, scalable architecture:

```graph
├── src/
│   ├── components/        # React + Astro components (includes ui/ primitives)
│   ├── layouts/           # Astro layouts (BaseLayout)
│   ├── lib/               # Shared utilities
│   ├── locales/           # Type-safe ES/EN dictionaries + locale helpers
│   ├── pages/             # Astro routes (/ , /en , /es)
│   ├── stores/            # Nanostores (theme/language)
│   ├── styles/            # Global styles (Tailwind + custom)
│   ├── test/              # Unit/component tests (Vitest)
│   └── types/             # Shared TypeScript contracts
├── tests/
│   └── e2e/               # End-to-end tests (Playwright)
├── public/                # Static assets
└── coverage/              # Coverage reports
```

## 🚀 Getting Started

To run this project locally, ensure you have **Node.js 20.19+** installed.

```bash
# 1. Clone the repository
git clone https://github.com/carlosdelcastillo/carlosdelcastillo.github.io.git

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) to view it in the browser.
